## 🚀 实际应用指南

### 📋 **部署建议**
```python

SFedAvgTrainer(
    d=feature_dim,
    r=int(0.5 * feature_dim),  # δ=0.5
    learning_rate=0.01,
    momentum=0.9,
    local_steps=5,
    client_fraction=0.3
)
```

- **高性能需求**: δ=0.5-1.0
- **平衡场景**: δ=0.5 (推荐)
